# Kafka Log Monitor - Backend

Kafka 기반 로그 수집 및 처리 시스템입니다.\
서비스 로그를 Kafka Topic으로 수집하고 Consumer에서 처리하여 DB에
저장하며, 실시간 로그 스트림을 제공합니다.

------------------------------------------------------------------------

## 기술 스택

-   Java 17
-   Spring Boot
-   Spring Kafka
-   Spring Data JPA
-   PostgreSQL
-   Docker

------------------------------------------------------------------------

## 주요 기능

### 로그 수집 API

서비스 로그를 Kafka Topic으로 전달합니다.

`POST /api/logs`

Kafka Producer를 통해 비동기로 로그를 전송합니다.

------------------------------------------------------------------------

### Kafka 기반 로그 처리

Kafka Consumer가 로그 메시지를 수신하여 DB에 저장합니다.

특징 - Batch Insert 처리 - Consumer Group 기반 병렬 처리

------------------------------------------------------------------------

### DLQ (Dead Letter Queue)

로그 처리 실패 시 DLQ Topic으로 메시지를 전송합니다.

Topic `log-events-dlq`

특징 - 장애 로그 분리 - 데이터 손실 방지

------------------------------------------------------------------------

### 로그 조회 API

로그 조회 및 검색 기능을 제공합니다.

`GET /api/logs/page`

지원 기능 - 로그 레벨 필터 - 서비스 필터 - 키워드 검색 - 페이징 조회

------------------------------------------------------------------------

### 오류 분석 API

서비스별 오류 발생 통계를 제공합니다.

`GET /api/logs/stats/errors`

------------------------------------------------------------------------

### 실시간 로그 스트림

Kafka Consumer 로그를 SSE(Server Sent Events)로 전달하여 실시간 로그
모니터링 기능을 제공합니다.

`GET /api/logs/stream`

------------------------------------------------------------------------

## 대용량 처리 전략

### Kafka Partition 분산

Kafka 메시지 Key를 다음과 같이 설계했습니다.

`serviceName + bucket`

예시

payment-0\
payment-1\
payment-2

특정 서비스 TPS 집중 시 Partition Hot Spot을 방지합니다.

------------------------------------------------------------------------

### Batch Insert

Kafka Consumer에서 로그를 Batch 단위로 DB에 저장하여 DB I/O를 줄이고
성능을 개선했습니다.

------------------------------------------------------------------------

### Index 설계

조회 성능 향상을 위해 다음 인덱스를 구성했습니다.

(service_name, created_at)\
(level)\
(created_at)

------------------------------------------------------------------------

### Retention 정책

오래된 로그를 자동 삭제하는 스케줄러를 구현했습니다.

LogRetentionScheduler

------------------------------------------------------------------------
