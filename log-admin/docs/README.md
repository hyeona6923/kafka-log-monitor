# Kafka Log Monitor - Admin Dashboard

Kafka 기반 로그 모니터링 시스템의 관리자 대시보드입니다.

로그 조회, 통계 분석, 실시간 로그 스트림을 제공합니다.

------------------------------------------------------------------------

## 기술 스택

-   React
-   TypeScript
-   Vite
-   Recharts

------------------------------------------------------------------------

## 주요 기능

### 로그 조회

로그 데이터를 페이징 기반으로 조회할 수 있습니다.

필터 기능

-   로그 레벨
-   서비스 이름
-   키워드 검색

------------------------------------------------------------------------

### 로그 통계

대시보드에서 로그 통계를 제공합니다.

-   레벨별 로그 통계
-   서비스별 로그 통계
-   서비스별 오류 분석

------------------------------------------------------------------------

### TOP 오류 서비스

서비스별 오류 발생 수를 분석하여 상위 서비스를 표시합니다.

------------------------------------------------------------------------

### 실시간 로그 스트림

SSE(Server Sent Events)를 활용하여 실시간 로그를 표시합니다.

예시

21:44:01 payment ERROR db timeout\
21:44:02 member WARN validation error\
21:44:03 auth INFO login success

------------------------------------------------------------------------

## 실행 방법

### 의존성 설치

npm install

### 개발 서버 실행

npm run dev

------------------------------------------------------------------------

## API 서버

http://localhost:8081

------------------------------------------------------------------------

## 화면 구성

Dashboard ├ 로그 통계 ├ 서비스별 오류 분석 ├ TOP 오류 서비스 ├ 실시간
로그 스트림 └ 로그 테이블
